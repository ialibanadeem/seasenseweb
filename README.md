# SeaSense AI - Full Ecosystem

SeaSense AI is an advanced maritime fleet intelligence platform. This repository contains the entire ecosystem, including the backend architecture, the Neo-Maritime dashboard, and mobile telemetry applications.

## 📂 Project Structure

- **[SeaSense-AI-dev](file:///e:/App/SeaSense-AI-dev)**: Core platform codebase.
  - **backend**: NestJS API with PostGIS and Redis integration.
  - **dashboard**: Next.js 15 "Neo-Maritime" command center.
- **[SeaSense-AI-MobileApps](file:///e:/App/SeaSense-AI-MobileApps)**: React Native / Expo telemetry app.
- **[maritime app](file:///e:/App/maritime app)**: Legacy or supplementary maritime components.

## 🚀 Key Features

- **Neo-Maritime Aesthetic**: A mission-critical, high-contrast UI with Neon Blue highlights.
- **Real-time Radar**: Sub-second vessel tracking and telemetry.
- **Fleet Intelligence**: Integrated hardware (ESP32) and mobile ingestion.
- **Mission Hub**: Emergency SOS monitoring and tactical command widgets.

## 🛠️ Quick Start

1. **Backend**:
   ```bash
   cd SeaSense-AI-dev
   npm install
   npm run start:dev
   ```

2. **Dashboard**:
   ```bash
   cd SeaSense-AI-dev/dashboard
   npm install
   npm run dev
   ```

3. **Mobile**:
   ```bash
   cd SeaSense-AI-MobileApps
   npx expo start
   ```
